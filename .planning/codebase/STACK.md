# Tech Stack

## Core Technologies
- **Language:** Dart 3.x
- **Framework:** Flutter (Android & iOS)
- **State Management:** Provider
- **Local Storage:** SQLite (via `sqflite` package)
- **Navigation:** Flutter SDK standard `Navigator`

## Key Packages & Dependencies
- `provider`: Dependency injection and state management.
- `sqflite`: Local database operations.
- `path`: File path manipulations.
- `intl`: Internationalization and date formatting.
- `google_fonts`: Custom typography (Outfit font).
- `flutter_localizations`: Localized support for Portuguese.

## Development & Build Tools
- **Build System:** Gradle (Android), CocoaPods (iOS).
- **Package Manager:** Pub.
- **Testing:** `flutter_test`.
