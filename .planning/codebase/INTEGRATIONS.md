# Integrations

## External Services
- None (currently a fully local, offline-first application).

## Infrastructure & API
- **Local Persistence:** All data is stored in a local SQLite database (`aventura_historia.db`).
- **Media Assets:** All character images and UI assets are bundled with the app as assets.

## Future Potential Integrations
- Cloud sync (Firebase/Supabase).
- Remote content updates for new historical eras.
- Achievements & Leaderboards (Play Games / Game Center).
